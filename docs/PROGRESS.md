## 🚀 Quick Commands Reference

### Development:

```bash
# Frontend
cd frontend && npm run dev

# Backend
cd backend && npm run dev

# Both (separate terminals)
npm run dev:all
```

### Testing:

```bash
# Build test
npm run build

# Production preview
npm run preview
```

### Deployment:

```bash
# Production build
docker-compose up -d --build

# Check logs
docker-compose logs -f

# Stop services
docker-compose down
```
